# Website Renewal Project — DUO (merry me)

**Project by**: jang Heui Ran  

## 제작기간

- **2023.04.10 ~ 2023.08.31 (22주)**

## Overview

`DUO`는 다양한 강점을 지닌 기업임에도 불구하고 구시대적인 디자인, 피로감을 유발하는 정보 구조로 인해 사용자가 웹사이트 이용을 꺼리게 되는 문제가 있었습니다.  
본 프로젝트는 **트렌디함과 설렘을 살린 톤앤매너로 리브랜딩**을 시도하고, **사용자 편의 중심의 구조 개편**을 통해 브랜드 이미지를 개선하고 잠재 고객 유입을 유도하는 것을 목표로 합니다.

## 문제인식

- **구시대적 비주얼**로 브랜드 인상이 약함
- **복잡하고 피로한 구조**로 원하는 정보 탐색이 어려움
- **사용자 이탈**로 이어져 잠재 고객 확보에 불리

## 해결방안

- **리브랜딩 기반 UI 리뉴얼**: 감성적인 컬러/이미지 톤 정리, 섹션별 메시지 명확화
- **GNB 중심의 동선 단순화**: 상단 메뉴에서 주요 페이지로 빠르게 접근
- **원페이지형 흐름 + 섹션 구성**: 메인에서 핵심 콘텐츠를 단계적으로 노출
- **반응형 대응**: 해상도별 스타일 분기 및 이미지 소스 대응
- **인터랙션 강화**: 슬라이더/호버/캐러셀 등으로 탐색 경험 개선

## 주요기능

### 현재 구현(이 저장소 기준)

- **직관적인 사용자 경험**
  - 상단 **GNB 메뉴**로 주요 페이지 빠른 이동
  - 드롭다운 서브 메뉴(hover)로 정보 탐색 보조

- **HOME(메인) 원페이지 구성**
  - **세로 스크롤 기반 섹션 전환(마우스휠)** 으로 흐름형 탐색 제공
  - 메인/정보/서비스/후기/상담소/예약/파트너 섹션 등 단계적 구성

- **슬라이더/캐러셀 인터랙션**
  - 섹션 내 **Swiper 슬라이드(탭/카드형 콘텐츠)** 제공
  - 파트너 로고 **무한 이동 캐러셀** 구현

- **반응형 UI**
  - 해상도별 CSS 분기(`main_768.css`, `main_360.css`)
  - 특정 섹션 배경 이미지 **디바이스 폭에 따른 소스 교체**

- **모바일 내비게이션**
  - 토글 버튼으로 메뉴 열기/닫기 제공(태블릿/모바일 대응)

- **상담 예약 UI**
  - 입력 폼 및 체크박스 동의 영역 구성(예약 CTA 섹션 제공)
